RegExp对象与正则表达式

# RegExp对象与正则表达式

RegExp对象提供正则表达式的功能

# 什么是表达式？

简单的表达式就是一个直接量、常量或变量
复杂的表达式使用运算符把一个或多个简单的表达式连接起来
不管表达式的形式多么复杂，表达式的功能是执行计算，最后都要求返回一个唯一的值。

# 什么是正则表达式？

正则表达式（regular expression）是一种表达文本模式（即字符串结构）的方法，常常用来按照“给定模式”匹配文本

新建正则表达式

// 法1：
var regx = /xyz/
// 法2：
var regx = new RegExp('xyz')

// 写法等价，都新建了一个内容为xyz的正则表达式对象

# 正则相关实例方法

# RegExp.prototype.test()

test()方法返回布尔值，表示参数字符串是否匹配，例：

let regx = /xyz/;
let res=regx.test('uvwxyz')
console.log(res); //true

# String.prototype.replace()

replace()方法可以替换匹配的值。它接受两个参数，第一个是正则表达式，表示搜索模式，第二个是替换的内容。例：

'aaa'.replace('a', 'b') // "baa"
'aaa'.replace(/a/, 'b') // "baa"
'aaa'.replace(/a/g, 'b') // "bbb"

# 字面量字符和元字符

  1. 字面量字符
    字符只表示它字面的含义，如 /a/ 匹配 a
  2. 元字符
    不代表字面的意思的部分特殊字符
    2.1 点字符 .
    匹配除 回车(\r)、换行(\n)、行分隔符、段分隔符以外所有字符
    如：/c.t/ 会匹配c和t之间包含任意一个字符的情况，只要这三个字符在同一行，比如cat、c2t、c-t等等，但是不匹配coot
    2.2 位置字符
    ^ 表示字符串的开始位置
    $ 表示字符串的结束位置
// test必须出现在开始位置
/^test/.test('test123') // true

// test必须出现在结束位置
/test$/.test('new test') // true

// 从开始位置到结束位置只有test
/^test$/.test('test') // true
/^test$/.test('test test') // false
