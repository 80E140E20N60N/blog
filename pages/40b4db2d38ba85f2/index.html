高级函数 reduce和map以及filter的区别

# 50.高级函数：reduce和map以及filter的区别

# 高阶函数

高阶函数是对其他函数进行操作的函数，它接收函数作为参数或将函数作为输出返回

# reduce和map以及filter的区别

# Array.prototype.reduce

该方法用于对数组中的每个元素执行一个 「reducer」 函数，并将结果汇总为单个返回值

// 累加
const src = [1,2,3]
let sum = src.reduce((acc,cur)=>{
    return acc+cur
},2)
console.log(sum) //8   

// 注：acc的初始值为传的参数2


// 数组去重
const src = [1,2,3,2]
let newArr=src.reduce((acc,cur)=>{
    acc.indexOf(cur)===-1&&acc.push(cur)
    return acc
},[])

console.log(newArr);

# Array.prototype.map

map()方法：

  1. map( )传入的参数是函数对象本身，该函数作用在Array的每一个元素并把结果生成一个新的Array返回
  2. map( )不会改变原始数组
var arr = [1, 2, 3, 4, 5, 6, 7, 8, 9];
var results = arr.map(item=>{
    return item*item
}); 
console.log(arr,results);
// [1, 2, 3, 4, 5, 6, 7, 8, 9]
// [1, 4, 9, 16, 25, 36, 49, 64, 81]

map()作为高阶函数，事实上它把运算规则抽象了,可以计算任意复杂的函数，比如，把Array的所有数字转为字符串：

var arr = [1,2,3,4,5]
let res=arr.map(String)
console.log(res); //[ '1', '2', '3', '4', '5' ]

# Array.prototype.filter

filter()方法：

  1. filter( )传入的参数是函数对象本身，该函数作用在Array的每一个元素，然后根据返回值是true还是false决定保留还是丢弃该元素。
  2. filter( )方法创建一个新数组，新数组中的元素是通过检查指定数组中符合条件的所有元素。其并不会改变原始数组
var arr = [1, 2, 4, 5, 6, 9, 10, 15];
var res =arr.filter(item=>{
    return item % 2 !== 0;
})
console.log(res); // [1, 5, 9, 15]

filter()接收的回调函数，其实可以有多个参数。通常我们仅使用第一个参数，表示Array的某个元素。回调函数还可以接收另外两个参数，表示元素的位置和数组本身：

var arr = ['A', 'B', 'C'];
var r = arr.filter(function (element, index, self) {
    console.log(element); // 依次打印'A', 'B', 'C'
    console.log(index); // 依次打印0, 1, 2
    console.log(self); // 数组自身 ['A', 'B', 'C']
});

# find()

var people = [
  { id: 1, name: '张三', age: 15, sex: 0, like: '睡觉' },
  { id: 3, name: '王五', age: 60, sex: 1, like: '下棋' },
  { id: 2, name: '李四', age: 35, sex: 1, like: '读书' },
]

let adult = people.find((item)=>{
  return item.age>18
})

let absent=people.find((item)=>{
  return item.age<15
})

let man=people.find((item)=>{
  return item.sex===1
})

man.like='跳广场舞'

console.log('不存在', absent)
console.log('成年人', adult)
console.log('男性', man)
console.log('原数组',people); 
 
/**
 * find()方法用来查找数组中符合条件的内容，当找到并放回符合条件的内容后，数组剩余的项不再执行find()方法
 * 
 * 如果找不到符合条件的项，返回undefined
 * 
 * find()方法不会开辟新的内存地址
