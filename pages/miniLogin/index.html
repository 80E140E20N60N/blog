    小程序登录授权

    # 小程序登录授权

    # 调用 wx.login 接口

    一般在onLoad()钩子函数中调用wx.login()， 获取临时登录凭证（code），并回传到开发者服务器。换取

    1. 用户在当前小程序的唯一标识 OpenID 、
    2. 用户在微信开放平台帐号下的唯一标识UnionID（若当前小程序已绑定到微信开放平台帐号）
    3. 本次登录的会话密钥 session_key。（用户数据的加解密通讯需要依赖会话密钥完成）

    # 获取手机号

    微信获取手机号，微信官方提供新旧两个版本的方法：

    旧版本接口指南 (opens new window)

    获取微信用户绑定的手机号，需先调用wx.login接口。因为需要用户主动触发才能发起获取手机号接口，所以该功能不由 API 来调用，需用 button 组件的点击来触发。

    注意：wx.checkSession 可以进行登录态检查，避免 login 刷新登录态。
