# Echarts常用配置

echarts配置项里面最主要的就三个属性。分别是xAxis、yAxis和series，以这三个属性为主干去拓展，以此看官方文档 (opens new window)

    const option = {
		//气泡提示框，用于配置鼠标滑过或点击图表时的显示框。
        tooltip: { },
        
        //图例，表述数据和图形的关联，比如描述什么颜色代表什么类型
        legend: { },
        
        //直角坐标系中除坐标轴外的绘图网格，用于定义直角系整体布局
        grid: { },
        
        xAxis: { }, 
        yAxis: { },
        
        //数据系列，一个图表可能包含多个系列，每一个系列可能包含多个数据
        series:[
			{ },
			{ }
		]
    }

# legend配置

 legend: {
	show: true, //是否显示
	type: "plain", // 图例的类型 'plain':普通图例  'scroll':可滚动翻页的图例

  }

    transformSize(res) {
      const docEl = document.documentElement
      const clientWidth = window.innerWidth || docEl.clientWidth || document.body.clientWidth
      if (!clientWidth) return
      const fontSize = 100 * (clientWidth / 1920)
      return parseInt(res * fontSize)
    },

# grid配置

通过配置grid组件，来调整图表在div容器里的位置，以及图表的背景样式

grid: {
   left: '5%',  //配置left,top,right,bottom的值即可调整图表的位置
   right: '5%',
   top: '15%',  //可以是相对于容器高宽的百分比如top:'15%'
   bottom: 0,   //也可以是具体像素值
   containLabel: true //设置containLabel: true常用于 防止标签溢出
   show: true,  //是否显示直角坐标系网格的开关
   backgroundColor:'#ccc' //网格背景色，默认透明。
}

# xAxis配置

直角坐标系 grid 中的 x 轴，一般情况下单个 grid 组件最多只能放上下两个 x 轴

重点需要关注三个属性：axisLine、axisLabel和axisTick

type：坐标轴类型

​ 1、'value' 数值轴，适用于连续数据; ​ 2、'category' 类目轴，适用于离散的类目数据，为该类型时必须通过 data 设置类目数据; ​ 3、'time' 时间轴，适用于连续的时序数据，与数值轴相比时间轴带有时间的格式化

xAxis: {	
    type: 'category',     //坐标轴类型：'value','category','time'
    axisLine: {                // x坐标轴轴线
       show:true               // 轴线是否显示
	   lineStyle: {
           color: 'rgba(255, 255, 255, .1)'
       }
    },
    axisTick: { show: false }, // x坐标轴刻度
    axisLabel: {               // x坐标轴刻度标签
       margin: 17,
       fontSize: 11,
       rotate,          //文本字段比较长,如果要全部显示出来就可以考虑用
       color: '#EBF5FF' //刻度标签文字颜色，支持回调函数
    },
    data: xAxisData,   //设置 x 轴上的值
	boundaryGap:true  //x坐标轴两边留白策略，true:x坐标轴两边留白,false:x坐标轴两边不留白,或者使用百分比灵活调整['30%', '20%']
}

# yAxis配置

同xAxis一样的是yAxis也需要关注axisLine、axisLabel和axisTick，这三个属性跟xAxis差不多

不同的是yAxis要多关注几个属性，分别是min、max和splitNumber

min：坐标轴最小值

max：坐标轴最大值

yAxis: {
    type: 'value',        //坐标轴类型：'value','category','time'
    axisLine: {                // y坐标轴轴线
       lineStyle: {
          color: 'rgba(255, 255, 255, .1)'
       }
    },
    axisTick: { show: false }, // y坐标轴刻度
    axisLabel: {               // y坐标轴刻度标签
       margin: 17,             // y轴标签距离图表的位置
       fontSize: 11,
       color: '#D6E7FF'
    },
    splitLine: {               // 分割线
       lineStyle: {
         color: 'rgba(255, 255, 255, .1)'
       }
    }
}

# series配置

每个系列通过type决定自己的图表类型，整个echarts的精髓就在于它

每种类型图表都有自己的特点，但是主要就是四点：'形状'、'大小'、'位置'、'颜色'

{
  name: '需求人数',
  type: 'line',
  smooth: true,
  data: valData,
  itemStyle: {
    normal: {
      color: '#007FFF', // 改变折线点的颜色
      lineStyle: {
        color: '#007FFF', // 改变折线颜色
        width: this.transformSize(0.02)// 线条粗细
      }
    }
  },
  areaStyle: {
    color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
      {
        offset: 0, color: '#007FFF' // 0% 处的颜色
      },
      {
        offset: 0.5, color: '#16436e' // 50% 处的颜色
      },
      {
        offset: 1.0, color: '#1e2852' // 100% 处的颜色
      }
    ])
  }
}

# formatter

使用formatter调用自定义的数据，把内容通过处理让变成我们想要的样子

# 字符串模板

模板变量有 {a}, {b}，{c}，{d}，{e}，具体如下：

类型 {a} {b} {c} {d}
折线（区域）图、柱状（条形）图、K线图 系列名称 类目值 数值 无
散点图（气泡）图 系列名称 数据名称 数值数组 无
地图 系列名称 区域名称 合并数值 无
饼图、仪表盘、漏斗图 系列名称 数据项名称 数值 百分比
// 饼图 自定义tooltip
  tooltip: {
    trigger: 'item',
    formatter: '{b}：{d}%'
  },

// 自定义axisLabel
// 坐标轴刻度上面的刻度格式化，比如Y表示长度，我们往往需要在每一个刻度值后面带上“m”的单位，Y轴意义不同，加不同的单位。
  yAxis: [
            {
                type: 'value',
                axisLabel: {
                    show:true,
                    formatter:'{value} m'
                },
            }
         ]

# 回调函数

自定义样式内容，例下：

// 修改tooltip的默认圆点样式，打印item，发现小圆点为marker字段的html，要改变小圆点颜色只需自定义html即可
  tooltip: {
    trigger: 'axis',
    axisPointer: {
      type: 'shadow'
    },
    formatter: function(params) {
      let tool = ''
      const numHtml = '<span style=\"display:inline-block;margin-right:5px;border-radius:10px;width:10px;height:10px;background-color:#0079fe;\"></span>'
      params.forEach(function(item) {
        if (item.componentSubType === 'bar') {
          tool += item.name + '</br>' + numHtml + item.seriesName + ':' + item.value
        } else {
          tool += '</br>' + item.marker + item.seriesName + ':' + item.value
        }
      })
      return tool
    }
  },

# datazoom

Echarts图表数据量过多时，可以使用datazoom来进行左右拖动展示，datazoom的配置可以根据数据量进行动态设置，当数据量超过一定值后再使用datazoom，例下：

 const option = {
   datazoom:labelData.length > 6 ? [
          {
            type: 'inside',
            start: 0,
            end: 15
          },
          {
            type: 'slider',
            height: 10,
            showDetail: false,
            showDataShadow: false,
            start: 0,
            end: 100,
            bottom: 0
          }
        ] : [],
 }
