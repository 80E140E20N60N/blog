浅拷贝与深拷贝

# 浅拷贝与深拷贝

# 浅拷贝

浅拷贝拷贝的是内存地址，当改变对象的值，那么浅拷贝出来的对象也会相应改变。

let obj1={ name: '张三', age: 25, address: '深圳',getName:(str)=>{console.log(str);}}
let obj2=obj1
obj1.name='李四'

console.log(obj1,obj2);
// { name: '李四', age: 25, address: '深圳', getName: [Function: getName] }
//  { name: '李四', age: 25, address: '深圳', getName: [Function: getName] }
//浅拷贝只能断开一层的引用，如果数据结构是多层对象的话，浅拷贝就不能解决问题了，这时候我们需要用到深拷贝。

# 深拷贝

深拷贝开辟了新的内存区域，当改变一个对象的值时，另一个对象的属性值并不受到影响

# 深拷贝方法一

JSON.parse(JSON.stringify())
用JSON.stringify将对象转成JSON字符串，再用JSON.parse()把字符串解析成对象，一去一来，新的对象产生了，而且对象会开辟新的栈，实现深拷贝。注：可以实现数组或对象深拷贝，但不能处理函数。

# 深拷贝方法二

递归浅拷贝

// 封装方法
function deepClone(source) {
    if (!source && typeof source !== 'object') {
      throw new Error('error arguments', 'deepClone')
    }
    const targetObj = source.constructor === Array ? [] : {}
    Object.keys(source).forEach(keys => {
      if (source[keys] && typeof source[keys] === 'object') {
        targetObj[keys] = deepClone(source[keys])
      } else {
        targetObj[keys] = source[keys]
      }
    })
    return targetObj
}
-------------------------------------------------------------------
let obj1={ name: '张三', age: 25, address: '深圳',getName:(str)=>{console.log(str);}}
let obj2=deepClone(obj1)
obj1.name='李四'

console.log(obj2,obj1);
//{ name: '张三', age: 25, address: '深圳', getName: [Function: getName] }
//{ name: '李四', age: 25, address: '深圳', getName: [Function: getName] }
